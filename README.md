# NASU Voice

Groq の Whisper API を使った Windows 向け音声入力ツールです。  
キーを押している間だけ録音し、離すと自動でテキストに変換してカーソル位置に貼り付けます。

## 特徴

- キーを**押している間だけ録音**、離したら即変換・貼り付け
- 初回起動時に **API キーとホットキーを GUI で設定**（設定ファイルの編集不要）
- Groq の Whisper large-v3-turbo を使用。**変換が非常に速い**
- API コストは月**数十円〜数百円程度**（無料枠あり）

## 必要なもの

- **Windows 10 / 11**
- **Python 3.9 以上**（[python.org](https://www.python.org/downloads/) からインストール）
- **Groq API キー**（無料で取得できます → [console.groq.com](https://console.groq.com)）

## インストール

```
pip install -r requirements.txt
```

## 起動方法

```
pythonw nasu_voice.py
```

または `start_nasu_voice.bat` をダブルクリックします。

**初回起動時：**
1. Groq API キーを入力する画面が出ます
2. 次に「録音に使うキーを押してください」と出るので、使いたいキーを押します
3. 設定は `nasu_config.json` に保存されます（次回から不要）

**おすすめのキー：**

| キー | 備考 |
|------|------|
| F9 / F10 / F11 / F12 | 最もおすすめ。他のアプリと干渉しにくい |
| ScrollLock / Pause | キーボードにあれば使いやすい |
| テンキーの各キー | テンキーがある場合に便利 |

⚠️ `Q` や `A` などの文字キーは避けてください。文字入力中に誤作動します。

## 使い方

1. ツールをバックグラウンドで起動しておく
2. テキストを入力したいアプリ（メモ帳・ブラウザ・チャットなど）にカーソルを置く
3. 設定したキーを**押している間**話す
4. キーを離すと自動で変換されてカーソル位置に貼り付けられる

## ホットキーの変更

`nasu_config.json` を開いて `"hotkey"` の値を `""` に変更してから起動すると、設定画面が再表示されます。

## 設定ファイル（nasu_config.json）

| 項目 | 説明 |
|------|------|
| `api_key` | Groq API キー |
| `hotkey` | 録音トリガーキー（初回設定後は自動で入力されます） |
| `language` | 言語コード（デフォルト: `ja`） |
| `prompt` | Whisper に渡すヒントワード（専門用語などをカンマ区切りで記入） |

## PC 起動時に自動で立ち上げる方法

1. `start_nasu_voice.bat` のショートカットを作成する
2. `Win + R` を押して `shell:startup` と入力して開く
3. ショートカットをそのフォルダに入れる

## Groq API の料金

無料枠：1日 2,000 リクエスト・1時間あたり 7,200 秒  
通常の使い方では無料枠を超えることはほとんどありません。  
無料枠を超えた場合は「本日の無料枠を使い切りました」と表示されてその日は使えなくなりますが、**自動的に課金されることはありません。**  
有料プランに自分でアップグレードしない限り、費用は一切かかりません。
